Long Short Term Memory
Created 2019-10-18| Updated 2019-10-24
Word count: 354|Reading time: 2 min|Post View:

The general Neural Network can only use independent data, it do not have the “memory”.

The first approach is to add a recurse in net. The output will still contain the information of the input. This called Recursive Neural Network (RNN), but it always leads to a Gradient Vanishing.

Here Ct−1C_{t-1}Ct−1​ and CtC_tCt​ is the Cell state to let information pass through with unchanged.

But for RNN, learning long-term dependencies with Gradient Decent is hard: http://ai.dinfo.unifi.it/paolo//ps/tnn-94-gradient.pdf

Then LSTM is designed as a special RNN and has 3 gate to control the cell state. It has 输入门，遗忘门，输出门.

Forget Gate

  • ht−1h_{t-1}ht−1​ is the output of the last cell(the whole green box above).
  • xtx_txt​ is the input of the current cell
  • σ\sigmaσ is the sigmod function: result is between 0 and 1, 0 means all forget and 1 means keep all.
  • ftf_tft​ is the result of the sigmoid function.

Input Gate

Sigmoid decide which information need to be update. iti_tit​ is the result of the sigmoid, so we can use it⋅Ct~i_t\cdot \tilde{C_t}it​⋅Ct​~​ to be the scaled value which means how much we decided to update each state value. Similarly, ft⋅Ct−1f_t\cdot C_{t-1}ft​⋅Ct−1​ is how much we want to keep from the previous state.

Tanh layer is to update the old information.

  1. CtC_tCt​ is the Cell information or Cell state. and Ct−1C_{t-1}Ct−1​ has been updated to CtC_tCt​
  2. Then we use ft⋅Ct−1+it⋅Ct~f_t\cdot C_{t-1} + i_t \cdot \tilde{C_t}ft​⋅Ct−1​+it​⋅Ct​~​ as our new CtC_tCt​

Output Gate

Based on our Cell information, we get the output hth_tht​ or the ht−1h_t-1ht​−1 for the next Cell.

  1. oto_tot​ is caculated by the sigmoid layer
  2. output hth_tht​ is computed by oto_tot​ times Cell information pass through the tanh layer (-1 to 1)
