Singular value decomposition

For Singular value decomposition, people always call it SVD. And it is a way to break up any m×nm\times nm×n (rectangular ) matrix into 3 pieces:

A=UΣVTA=U\Sigma V^{T} A=UΣVT

Where

UUU is a m×mm\times mm×m orthogonal matrix

Σ\SigmaΣ is a m×nm\times nm×n diagonal matrix, with singular value

VTV^{T}VT is a n×nn\times nn×n orthogonal matrix

Noted that we can rewrite any square matrix into:

B=VΛVTB=V\Lambda V^T B=VΛVT

Where VVV and VTV^TVT is the eigenvalue vector for matrix BBB and $\Lambda $ is the diagonal matrix with eigenvalue from top-left to bottom-right as large value to small value.

What is the difference between this two methods:

Let’s see

ATA=(VΣTUT)(UΣVT)=VΣTΣVTA^T A=(V\Sigma^T U^T)(U\Sigma V^{T})=V\Sigma^T \Sigma V^T ATA=(VΣTUT)(UΣVT)=VΣTΣVT

Since UUU is the orthogonal matrix, UTU=IU^TU=IUTU=I

And we can find the relation is that ATAA^T AATA is a square matrix and is positive definite since it is a square of a matrix AAA . So the eigenvaues are greater or equal than 0. VVV and VTV^TVT is the eigenvalue vector for ATAA^TAATA and ΣTΣ\Sigma^T \SigmaΣTΣ is the λ\lambdaλ (eigenvalue) for ATAA^TAATA or the σ2\sigma ^2σ2 for the ATAA^TAATA.

(Determinal is 0)= singular matrix, orthogonal matrix has determinant as 1, the product of Σ\SigmaΣ is same as the determinant of AAA
