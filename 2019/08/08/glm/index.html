Generalized Linear Model
Created 2019-08-08| Updated 2019-09-05
Word count: 655|Reading time: 4 min|Post View:

Objective:

  • General linear model (GLM) used to predict response variables with both continuous or discrete distribution and response variables can have linear or non-linear relationship with explanatory variables.

Model structure:

  • GLM includes three parts. Random component, Systematic component, Link function.

Assumption:

  • yiy_iyi​ is independent and typically assumed to follow an exponential family distribution with μi\mu_iμi​.
  • GLM assumes transformed dependent variables (through link function) and independent variables has a linear relationship.
  • Error ϵi\epsilon_iϵi​ should be independent but not normally distributed.

Parameter estimate:

  • xix_ixi​ as covariates and β\betaβ for coefficients

Model selection:

  • feature selection

Model fit:

  • Least square

Introduction of 3 components of any GLM:
random component: which is the probability distribution of response variable Y

systematic component: show the explanatory variables (x1,x2,..,xk)(x_1,x_2,..,x_k)(x1​,x2​,..,xk​) in the model. And GLM has a strong assumption that η=βx\eta=\beta xη=βx, which means the natural parateter η\etaη in exponential family should equal to linear predictor (design choice).

link function: η\etaη or g(u)g(u)g(u) - refer to the link between random and systematic components. In another words, it shows how response variables change based on explanatory variables. It can be a non-linear function. In addition, the inverse function g−1(η)=μg^{-1}(\eta)=\mug−1(η)=μ is the response function which can reflect linear predictor to the target yyy.

Advantage of GLM over OLS(ordinary least square)

  • There is no need to transform the respose YYY to have normal distribution.
  • Variance no need to be constant
