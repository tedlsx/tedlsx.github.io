Binomial distribution

For example, flip a coin for 5 times, define random variable X as number of heads after 5 flips. The probability is defined as:

p(X=n)=5!n!(5−n)!=C5np(X=n)=\frac{5!}{n!(5-n)!}=C_5^n p(X=n)=n!(5−n)!5!​=C5n​

The general probability is

p(X=k)=n!k!(n−k)!=Cnkp(X=k)=\frac{n!}{k!(n-k)!}=C_n^k p(X=k)=k!(n−k)!n!​=Cnk​

E(X)=np=∑k=0nkCnkpk(1−p)n−kE(X) = np=\sum_{k=0}^{n}kC_n^kp^k(1-p)^{n-k} E(X)=np=k=0∑n​kCnk​pk(1−p)n−k

Poisson distribution

For example, we want to know X = how many car pass in an hour.

Say E(X)=λ=npE(X) = \lambda = npE(X)=λ=np, where λ\lambdaλ has unit cars/hour ⇒60min/hourλ60cars/min\Rightarrow 60min/hour \frac{\lambda}{60} cars/min⇒60min/hour60λ​cars/min

Hence the probability can be written as whether there is a car pass in that minute which follows binomial distribution

p(X=k)=C60k(λ60)k(1−λ60)60−kp(X=k)=C_{60}^k(\frac{\lambda}{60})^k(1-\frac{\lambda}{60})^{60-k} p(X=k)=C60k​(60λ​)k(1−60λ​)60−k

But the question is if there are more than 1 car pass in a minute, how do we deal with it? The most straight way is to set minutes into second
